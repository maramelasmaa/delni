# Laravel Superpowers

Adds Laravel-aware code generation, migration/model/controller scaffolding, and artisan command help.

**Source:** GitHub – jpcaparas/superpowers-laravel

## Features

- **Code Generation**: Intelligent scaffolding for models, migrations, and controllers
- **Migration Assistance**: Generate database migrations with proper structure
- **Model Scaffolding**: Create models with relationships and factories
- **Controller Generation**: Generate RESTful controllers with type hints
- **Artisan Help**: Quick access to Laravel artisan commands and their parameters

## Usage

This tool integrates with Claude Code to provide Laravel-specific code generation shortcuts. Use it when:

- Creating new models with relationships
- Generating database migrations for new features
- Scaffolding controllers for API endpoints
- Setting up factories and seeders
- Exploring available artisan commands
