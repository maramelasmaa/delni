INFRASTRUCTURE NOTES
================================================================================

[✅] Scheduler (Auto-Expiry) Configuration
    Required cron job:
    * * * * * cd /var/www/delni && php artisan schedule:run >> /dev/null 2>&1
    
    This is CRITICAL for subscription expiry at 00:00 UTC

[✅] Queue Workers
    Delni uses queued jobs for:
    - Profile creation on user signup
    - Profile stats recalculation on review changes
    
    NOT used for: profile deletion/cleanup

[✅] Deployment Procedure
    1. php artisan down
    2. git pull
    3. composer install --no-dev --optimize-autoloader
    4. php artisan migrate --force
    5. php artisan optimize:clear
    6. php artisan optimize
    7. php artisan queue:restart
    8. php artisan up
